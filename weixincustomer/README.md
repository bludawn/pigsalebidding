# 生猪贸易软件系统——微信小程序客户端

本工程是项目微信小程序客户端，给企业客户采购生猪使用，使用nodejs开发，开发语言是typescript。

软件功能页面分为3大部分，分别是商品列表清单、我的、消息中心；
- 商品列表清单：有筛选排序、查看详情、列表信息
- 我的：有个人信息、我的订单、收货地址、总资产、功能区、交易总金额、购买总头数、设置
- 消息中心：有通知

## 本工程必须依赖的规范参考 CODERULE.md 文件

## 本地运行

**需要:**  Node.js

1. 安装依赖:
   `npm install`
2.  dev运行:
   `npm run dev`
